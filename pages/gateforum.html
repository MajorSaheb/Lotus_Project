Online TarGATE

Online TarGATE is a complete online assessment module for GATE Aspirants. Apart from giving you practice in high quality questions and a comprehensive comparative analysis (AIR, Percentile, Score, section-wise performance analysis etc), also gives you practice of taking a test in exact GATE like conditions (monitored tests).

Online TarGATE Features:
Online Test Series
12 Self Administered Tests (Full Syllabus), conducted according to a schedule, with detailed performance analysis after each SAT (All India Rank, Percentile, Score, section wise analysis etc)40 Section Tests (entire section syllabus)40 unit tests (partial section syllabus)User interface and pattern is very similar to actual Online GATE (with numerical and multiple choice questions)

Score Maximizer
4 Monitored Tests - MTs (Full Syllabus) to be attempted at GATEFORUM designated computer labs, under exact GATE like conditions.10 Extra Edge Tests - EETs (Full Syllabus)Video lectures of Engineering Mathematics and General Ability, GATE Solutions (Online Access)

Important Note:
●Self Administered Tests:Each Self Administered Test will have a scheduled start date and end date. The results of each test will be computed after the window period of that particular test. After the window period, a test can be attempted but a detailed performance analysis like All India Rank, Percentile; Score etc will not be given (only total marks scored, Number of Correct & incorrect answers, section wise marks scored etc will be given if a test is attempted after the end date).
●Monitored Tests: MTs (Monitored test) will have to be taken in a particular slot at a venue allotted by GATEFORUM. MTs (Monitored test) can be attempted after the end date but the rules remain the same as for SATs (Self administered test).
●Section Tests:will not have any end date. After attempting the section tests, marks scored, total number of correct & incorrect answers will be given.
●Unit Tests: Unit tests cover partial sections and can be taken from any computer via internet.The results of the Extra Edge Test will be updated every week. Therefore you can take the tests anytime according to your convenience and still get your All India Rank, Percentile, Score etc.
